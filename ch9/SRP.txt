컴포넌트의 주요 역할을 정확히 파악해서 하나만 부여
아래의 기술로 적용

1. render props : props를 이용하는 기술, 렌더링을 위임
2. 합성 : 작고 재사용 가능한 컴포넌트를 만들고 이를 조합해서 더 복잡한 UI 구성

------------------------------------------------------------------------------------

1. render props: children이나 기타 props로 렌더링 될걸 children에 넣어서 잘 처리

!! children에 함수를 넣는 예제 !!

const Title = ({ title, children } : { title, children }) => {
  return <div>{children(title)}</div>
}

<Title>
  {(s: string) => {
    const formatted = s.toUpperCase();
    
    return <h3>{formatted}</h3>
  }}
</Title>

부모에서 자식 컴포넌트의 렌더링을 제어할 수 있게 된다.
